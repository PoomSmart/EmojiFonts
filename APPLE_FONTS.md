## `AppleColorEmoji-160px.ttc` (for iOS 10+)
Compressed, 160x160 image set included.
For iOS 10-13, place the downloaded file at `/System/Library/Fonts/Core` and rename that file to `AppleColorEmoji@2x.ttc`.
For iPhone SE 1st gen, iPhone 6s and iPhone 6s Plus, place the downloaded file at `/System/Library/Fonts/Core` and rename that file to `AppleColorEmoji.ttc`.
Otherwise, place the downloaded file at `/System/Library/Fonts/CoreAddition`.

## `AppleColorEmoji@2x.ttc` (for iOS 10 - 13)
Place the downloaded file at `/System/Library/Fonts/Core`.

## `AppleColorEmoji@2x.ttf` (for iOS 5 to 9)
For iOS 8.2 to 9.3, place the downloaded file at `/System/Library/Fonts/Core`.
For iOS 5.1 to 8.1, place the downloaded file at `/System/Library/Fonts/Cache`.

## Remarks

Refer to your default emoji font location from [this table](https://poomsmart.github.io/emojiport#fontlocations):

<img width="1320" alt="image" src="https://user-images.githubusercontent.com/3608783/219857015-6cb7bd1a-8922-42f4-82ed-0e921b824ccc.png">

If you are on iOS 10 or higher, download either `AppleColorEmoji-160px.ttc` or `AppleColorEmoji@2x.ttc`. The `AppleColorEmoji-160px.ttc` one includes 160x160 emoji images.

If you are on iOS 9 or lower, download `AppleColorEmoji@2x.ttf`.

Once you downloaded the font from here, rename it to match with the default emoji font name found from the table above then copy it to the path indicated by that same table.

You may use [this UI](https://poomsmart.github.io/emojiport#emoji-font-finder) to locate your emoji font path.
